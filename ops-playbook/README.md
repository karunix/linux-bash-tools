# Linux Bash Operations Playbook

A modular toolkit of lightweight Bash scripts and playbooks for Linux system administration, security auditing, and operational troubleshooting.

## Purpose

Provide simple, repeatable command-line tools for:

- System health checks
- User auditing
- SSH abuse detection
- Pre-maintenance validation
- Docker inspection (future)

## Philosophy

- Simple not over engineered
- Fast execution
- Minimal dependencies
- Clear output
- Expandable through playbooks

## Structure

Each module contains:
- Script
- README (usage + purpose)
- Clear operational context

Built for practical sysadmin use.
