# System Baseline Playbook

## Purpose
Quickly assess overall health and exposure of a Linux system.

## When to Run
- New server onboarding
- Before hardening
- After incident
- Before deploying applications

## What To Check
- High load average
- Memory pressure
- Disk >80%
- Unknown listening ports
- Unexpected high CPU processes

## Follow-up Actions
Investigate suspicious processes and open ports immediately.
